# DailyNews
一款基于开源新闻数据的新闻类APP。  

### 一、简介
DailyNews是一款新闻资讯类App,主要包含内容：
  
1. 主页（知乎热搜、果壳精选、豆瓣一刻）
2. 干货（每日推荐，干货定制，福利）
3. 电影（热门电影、Top250电影推荐）
4. 个人（头像设置，设置）

使用MaterialDesign+Mvp+Retorfit2+Rxjava2+Glide+Rxbus等最新的Android开发技术来进行开发。对自己所学的Android知识进行汇总,总结。
